Parking Manager Application for Mall:

Parking Managing app - A console based python application that lets admin to assign multiple staffs as a parking managers and let the parking managers to add customer's vehicles  to parking and un-park them with the price calculation with respect to the time and also let them to lock the vehicles when exceeded the time limit. This app also let the customers to see their vehicle's status in parking. This app involves the all CRUD operations such as Creation/Adding of parking managers/vehicles, Reading of vehicle's status, Updation of vehicle's parking status and Deleting of Parking manager.JSON files are used to store the application data, making the app faster and compact. 

Concepts covered - Function, Date and time module, Regular Expression (Regex) and JSON File Handling.

Tools used - Python 3.8, VS Code.

Database: Non-Relational Database (JSON document Database)
